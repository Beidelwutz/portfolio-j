# Projekt-Bilder

Hier die Bilder für die Detailseiten ablegen:

## Projekt Alpha
- `alpha-thumb.webp` – Thumbnail für Featured-Card
- `alpha-hero.webp` – Hero-Bild (ca. 1920x1080)
- `alpha-01.webp` bis `alpha-05.webp` – Galerie-Bilder

## Projekt Beta
- `beta-thumb.webp` – Thumbnail für Featured-Card
- `beta-hero.webp` – Hero-Bild
- `beta-01.webp` bis `beta-06.webp` – Galerie-Bilder

## Projekt Gamma
- `gamma-thumb.webp` – Thumbnail für Featured-Card
- `gamma-hero.mp4` – Hero-Video (optional)
- `gamma-01.webp` bis `gamma-07.webp` – Galerie-Bilder

**Empfohlene Formate:**
- Thumbnails: 800x600px, WebP
- Hero: 1920x1080px, WebP oder MP4
- Galerie: Verschiedene Größen, WebP
