# Uploads – Medien pro Bereich

Bilder und Vorschaubilder für die Portfolio-Bereiche:

- **videos/** – Vorschaubilder/Poster für Video-Projekte
- **clips/** – Vorschaubilder für Clip-Projekte  
- **fotos/** – Bilder für Foto-Projekte

Dateien unter `public/uploads/` sind z.B. unter `/uploads/videos/…` erreichbar.
